# Daily Planner

A fully offline daily planner app built with React Native, Expo, and NativeWind.

## Features

- **Today** — View and manage today's scheduled tasks with progress tracking
- **Progress** — Browse history by date, view analytics, streaks, and weekly averages
- **Edit Tasks** — Create and edit tasks with time ranges, completion %, and categories
- **Settings** — Customize your name and view categories
- **Offline-first** — All data stored locally in SQLite via `expo-sqlite`

## Getting Started

```bash
npm install
npx expo start
```

Press `i` for iOS simulator, `a` for Android emulator, or scan the QR code with Expo Go.

## Tech Stack

- React Native + Expo
- NativeWind (Tailwind CSS)
- React Navigation
- expo-sqlite (local persistence)
- lucide-react-native (icons)
